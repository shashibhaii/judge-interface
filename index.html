Judge-Interface

This is the default interface for openjudge.

Setup

$ cd ~
$ git clone https://github.com/theSage21/judge-interface.git
$ cd judge-interface
$ ./setup.sh

Usage

$ cd ~
$ cd judge-interface
$ source env/bin/activate
$ ./runserver.sh

The Django admin provides:

  • Add participants
  • Add Questions and Answers
  • Add languages
